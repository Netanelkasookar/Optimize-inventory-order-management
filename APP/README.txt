This software will calculate a new ordering point by using the Monte Carlo simulation regardless of service levels.

Instructions
1. Create folder on drive C with the name "App" (Path: C:\App).
2. Transfer Excel file file name with the data according to the format.
3. Run the software "Calculate order point.exe".
4. Selecting a number of simulations to run (Any integer can be selected but recommended between 100 and 10,000).
5. After the software has finished, a new file will be created for the "Application" name of the result containing the new order point values.
